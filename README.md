# upnext

upnext is a self-hosted, multi-platform task tracker designed to help you stay organized without the overhead of accounts, cloud services, or complex setup.

## Architecture & Components

### Backend
 - Postgres Database - Single database for data persistence. 
 - ASP.NET Server - Core application server and data access layer.
